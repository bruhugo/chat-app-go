# Real time chat application in Golang 

This is the backend repository for my real time chat app. 

## Features

- Message/User/Chat CRUD endpoints
- Websocket connections
- Simple auth flow with single JWT token
- Api documentation with Swagger

## Stack 

- Golang
- HTTP
- MySQL
- Swagger
- Gin
- Terraform
- Gorilla Websocket

## Infrastructure

Currently, I'm using ECS to host the containers and RDS MySQL database as described in the infra directory.
